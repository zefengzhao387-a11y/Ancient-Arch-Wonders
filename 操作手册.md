# Ancient Arch Wonders（古建筑奇境）操作说明

本说明供评委在试玩时对照使用。作品英文名为 Ancient Arch Wonders；叙事场景多按传统院落空间搭建，知识点侧重古建筑修缮中的通用概念，与具体景区宣传无关。

## 一、运行环境

- **系统**：Windows 10 / 11，64 位  
- **外设**：键盘、鼠标  
- **显示**：建议 16∶9；工程内已做黑边适配，非 16∶9 可能出现左右或上下黑边，属预期表现  
- **时长**：完整体验约十余至二十分钟；**无中途存档**，退出后需从主菜单重新开始。答辩前建议在目标机器上通跑一遍。

## 二、操作一览

| 用途 | 默认键位 |
|------|----------|
| 界面点击、对话「继续」、榫卯拖拽 | 鼠标左键 |
| 左右移动 | A、D（户外、第二章平台关、第三章桥上通用） |
| 跳跃 | W、空格；第二章平台关亦支持方向键 **↑** |
| 交互（进门、进剧情等） | E |
| 第二章出口确认 | 需先收集足够场景构件（默认 3 个），角色站在出口区域内，再按 **S** |
| 第三章罗盘修复模式 | 按界面提示，用方向键 **↑ / ↓** 微调指针 |
| 榫卯测量「彻底铆合」 | 指针进入中间**绿色区域**后按 **Enter**；若工程内改过键，以游戏内提示为准 |

榫卯关：**须在当前槽完成测量并回车铆合后**，再操作其他槽；否则将弹出提示并禁止插队，以免打乱教学顺序。

## 三、主线流程（概览）

主菜单 **开始** → 开场（静图 / 序列帧 / 短视频与黑场组合，部分步骤需鼠标点击推进）→ 第一章对话图 → 户外行至门口 **E** → 规则视频 → 榫卯三槽全部完成 → 第一章收束（短视频与对白）→ 第二章（标题、进门、卷轴、平台关等，按场景提示操作）→ 第二章通关视频与小知识图 → 第三章卢沟桥（行走、对话、**E** 进入罗盘）→ 结尾多段图与视频 → 返回主菜单。

具体关卡跳转以**当前打包版本**中场景配置为准。

## 四、常见问题

- **平台关无法离开**：多为构件未捡齐，或角色未站在出口判定区域内。  
- **罗盘**：先看屏幕文字；修复模式注意倒计时，失败可重试。  
- **场景切换时短暂黑屏**：为全屏黑色遮罩过渡，属正常表现。  
- **某一关长时间黑屏无响应**：可能为视频未正确加载，可结合《运行说明》中的日志路径，在 `Player.log` 中检索 `VideoPlayer`、`Exception` 等关键字辅助判断。

## 五、运行日志路径

默认示例（用户名、公司名、产品名若于 Unity Player Settings 中修改，对应文件夹名会随之变化）：

`C:\Users\<用户名>\AppData\LocalLow\DefaultCompany\Ancient Arch Wonders\Player.log`

也可在资源管理器地址栏输入 `%USERPROFILE%\AppData\LocalLow`，再按实际公司名 / 产品名逐级进入。
